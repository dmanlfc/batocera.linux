# Batocera Linux Distribution Policy

## Purpose

This document provides guidance for redistribution of Batocera Linux while preserving clarity for users and respect for contributors.

---

## Open Source Rights

Batocera Linux is distributed under GPLv3 and other compatible open-source licenses.

Redistribution rights are governed by those licenses.

Nothing in this document modifies or restricts those rights.

---

## Redistribution of Official Releases

Redistributors may:

- Mirror official downloads
- Archive official releases
- Include official releases in collections or repositories
- Provide access to official releases

Redistributors should clearly identify the source of official releases.

---

## Modified Distributions

Modified versions should:

- Clearly identify themselves as modified
- Clearly identify themselves as unofficial
- Avoid creating confusion regarding project affiliation
- Document significant changes where practical

---

## User Clarity

Redistributors should ensure users understand:

- Whether a release is official or unofficial
- Whether modifications have been applied
- Where support should be obtained

---

## Commercial Redistribution

GPLv3 permits commercial redistribution.

The Batocera Linux Project encourages commercial users and redistributors to contribute improvements, bug fixes, documentation, or financial support back to the ecosystem whenever practical.

---

## Community Stewardship

The project encourages redistribution practices that strengthen the ecosystem and support the long-term sustainability of Batocera Linux.
